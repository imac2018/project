Compilation:
make

Lancement:
./QtOpenGL

Jeu:
Trouvez un certains nombre de duff (canette de bière) et allez
voir Apu (le responsable de Kiwk-I-Mac)
    -  3 duff pour le premier niveau
    -  4 duff pour le deuxièmre niveau

Vous pouvez interagir avec les objets, s'ils brillent lorsque
vous passez votre souris dessus, cliquez pour déclencher l'action
liée à l'objet.

Attention Flanders (un sympathique moustachu) peut venir vous
embêter, ne le laissez pas vous embarquer dans une conversation,
cliquez plusieurs fois sur lui pour le faire partir.

Vous pouvez à tout moment quitter le jeu en cliquant sur "Exit"
