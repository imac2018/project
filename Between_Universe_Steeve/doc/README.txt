How To play:
	Controls:
		Common:
			P : Break
			Escape : Config window

		Player 1:

			Move :
				Left Arrow : go backward
				Right Arrow : go forward
				Down Arrow : go bottom
				Up Arrow : go top
			
			Shoot :
				Space : main Weapon
				
		Player 2:
			
			Move :
				Numpad 4 : go backward
				Numpad 6 : go forward
				Numpad 5 : go bottom
				Numpad 8: go top
			
			Shoot :
				Mouse Right Button : main Weapon
				
		Player 3:
		
			Move :
				Numpad Z : go backward
				Numpad S : go forward
				Numpad Q : go bottom
				Numpad D: go top
			
			Shoot :
				Ctrl : main Weapon
	
	Items:
		"UP" = weapon level up
		Purple Ellipse = shield

	Enemy:
		- Fish
		- Green Rect
		- Green Ellipse
